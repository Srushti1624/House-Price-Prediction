# House-Price-Prediction
# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview

This project predicts house prices using **Linear Regression**, one of the most popular machine learning regression algorithms. The model is trained on a housing dataset containing features such as house size, number of bedrooms, and location.

The project demonstrates the complete machine learning workflow:

* Data Loading
* Data Exploration
* Missing Value Handling
* Feature Scaling
* Train-Test Split
* Model Training
* Prediction
* Model Evaluation

---

## 🚀 Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

---

## 📂 Dataset Features

* Size (Square Feet)
* Number of Bedrooms
* Location
* House Price (Target)

---

## ⚙️ Machine Learning Workflow

1. Load/Create Dataset
2. Explore Data
3. Handle Missing Values
4. Normalize Features
5. Split Dataset into Training and Testing Sets
6. Train Linear Regression Model
7. Predict House Prices
8. Evaluate using Mean Squared Error (MSE)

---

## 📊 Evaluation Metric

* Mean Squared Error (MSE)

---

## 📈 Output

The project displays:

* House Price Distribution
* Actual vs Predicted House Prices
* Mean Squared Error
